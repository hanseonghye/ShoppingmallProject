# /admin/category/{id or name} 

## 카테고리 수정

■ url

 `put : /admin/category/{id or name}`

■ request

- path
  

`{id or name}` : 해당 카테고리 pk 값이나 카테고리 명

- request body : json

  ```json
  {
    "name": "카테고리 명",
    "parent": 부모 카테고리 ID
  }
  ```

■ response 

- 성공 : status code `200`

  ```json
  {
      "result": "success", 
      "message": null, 
      "data":{
        "name": "카테고리 명",
        "parent": 부모 카테고리 ID
      }
  }
  ```

- 실패 : status code `400`

  ```json
  {
      "result":"fail",
      "message":"실패 원인",
      "data":null
  }
  ```



## 카테고리 삭제

■ url

 `delete : /admin/category/{id or name}`

■ request

- path

  `{id or name}` : 해당 카테고리 pk 값이나 카테고리 명

■ response

- 성공 : status code `200`

  ```json
  {
      "result": "success", 
      "message": null, 
      "data":"ok"
  }
  ```

- 실패 : status code `400`

  ```json
  {
      "result":"fail",
      "message":"실패 원인",
      "data":null
  }
  ```
