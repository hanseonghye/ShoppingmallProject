# /admin/category

## 전체 카테고리 조회

■ url

 `get : /admin/category`

■ request



■ response

- 성공 : status code `200`

  ```json
  {
      "result": "success", 
      "message": null, 
      "data":[{ //카테고리 list
          "name": "카테고리 명",
          "parent": 부모카테고리 ID
      },]
  }
  ```

- 실패 : status code `400`

  ```json
  {
      "result":"fail",
      "message":"실패 원인",
      "data":null
  }
  ```



## 카테고리 추가

■ url

 `post : /admin/category`

■ request

- request body : json

  ```json
  {
    "name": "카테고리 명",
    "parent": 부모 카테고리 ID
  }
  ```


■ response

- 성공 : status code `201`

  ```json
  {
      "result": "success", 
      "message": null, 
      "data":{
        "name": "카테고리 명",
        "parent": 부모 카테고리 ID
      }
  }
  ```

- 실패 : status code `400`

  ```json
  {
      "result":"fail",
      "message":"실패 원인",
      "data":null
  }
  ```
