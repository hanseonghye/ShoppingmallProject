# /users

## 회원 가입

■ url

 `post : /users`

■ request

- request body : json

  ```json
  {
      username : "이름",
      user_id : "아이디",
      password : "비밀번호",
      email : "이메일",
      phone_number : "전화번호"
  }
  ```


■ response

- 성공  : status code `201`

  ```json
  {
      "result": "success", 
      "message": null, 
      "data":{ // 생성된 user 정보
          username : "이름",
          user_id : "아이디",
          password : "비밀번호",
          email : "이메일",
          phone_number : "전화번호"
      }
  }
  ```

- 실패 : status code `400`

  ```json
  {
      "result":"fail",
      "message":"실패 원인",
      "data":null
  }
  ```
