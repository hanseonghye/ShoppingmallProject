# /users/{id or user_id}

## 회원 정보 조회

url

 `get : /users/{id or user_id}`

■ request

- path

  `{id or user_id}` : user_id나 id

■ response

- 성공 : status code `200`

  ```json
  {
      "result": "success", 
      "message": null, 
      "data":{ // 회원 정보
          username : "이름",
          user_id : "아이디",
          password : "비밀번호",
          email : "이메일",
          phone_number : "전화번호"
      }
  }
  ```

- 실패 : status code `400`

  ```json
  {
      "result":"fail",
      "message":"실패 원인",
      "data":null
  }
  ```
