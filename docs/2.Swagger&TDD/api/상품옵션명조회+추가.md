# /admin/products/{id}/options

## 상품 옵션명 조회

■ url

 `get : /admin/products/{id}/options`

■ request

- path

  `{id}` : 상품 id 값

■ response

- 성공 : status code `200`

  ```json
  {
      "result": "success", 
      "message": null, 
      "data":[{
          "name": "옵셩명",
          "product": 상품 id
      },]
  }
  ```

- 실패 : status code `400`

  ```json
  {
      "result":"fail",
      "message":"실패 원인",
      "data":null
  }
  ```



## 상품 옵션 추가

■ url

 `post : /admin/products/{id}/options`

■ request

- request body : json

  ```json
  {
    "name": "옵션명",
    "product": 상품 ID
  }
  ```

■ response

- 성공 : status code `201`

  ```json
  {
      "result": "success", 
      "message": null, 
      "data":{
        "name": "옵션명",
        "product": 상품 ID
      }
  }
  ```

- 실패 : status code `400`

  ```json
  {
      "result":"fail",
      "message":"실패 원인",
      "data":null
  }
  ```
