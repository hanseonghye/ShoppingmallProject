# /products/{id}

## 상품조회

■ url

 `get : /products/{pk}`

■ request

- path

  `{pk}` : 해당 상품의 pk값

■ response

- 성공 : status code `200`

  ```json
  {
      "result": "success",
      "message": null,
      "data": {
          "name": 상품명,
          "price": 대표가격,
          "file_url": 파일 url,
          "is_stock" : 재고 사용 여부,
          "image_url": 대표이미지 url,
          "category": 카테고리 pk,
          "product_details": [
              {
                  "price": 가격,
                  "stock": 재고량,
                  "option_no": 옵션 번호,
                  "option_name": 옵션명
              },
  				...
          ]
      }
  }
  ```

- 실패 : status code `400`

  ```json
  {
      "result":"fail",
      "message":"실패 원인",
      "data":null
  }
  ```
